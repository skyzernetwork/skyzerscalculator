## Overview
Skyzers Calculator is a **recreated Apple-style calculator** built as a learning project to explore how simple yet functional applications can be created using just **HTML, CSS, and JavaScript**.  

This version includes the **basic arithmetic functions**:
- Addition (`+`)  
- Subtraction (`-`)
